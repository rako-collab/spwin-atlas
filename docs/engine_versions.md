# SPWIN Engine Versions

## SPWIN v2.4 Platinum Pro

Purpose: pre-match betting decision support.

Key rules:

- No subjective upgrades
- Separate win probability from bet quality
- Market value is mandatory
- Trap detector can downgrade but not be ignored
- PASS is valid

## SPWIN-Live v2.4

Purpose: live/in-play analysis only.

Inputs include:

- Score and minute
- xG
- Shots and shots on target
- Corners
- Possession
- Red cards
- Substitutions
- Odds movement

## SPWIN v2.5 Platinum Pro

Purpose: pre-match engine with enhanced objective intelligence.

New modules include lineup confirmation, market shock, market consensus, tactical matchup, finishing efficiency, CLV, and portfolio optimization.

## SPWIN v3.0

Purpose: evidence-driven engine powered by SPWIN Atlas historical data.

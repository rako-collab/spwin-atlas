# SPWIN Scoring Methodology

## SPWIN v2.4 Baseline Formula

| Module | Weight |
|---|---:|
| Team Strength | 25 |
| Current Form | 20 |
| Market Value / EV | 20 |
| Motivation / Context | 15 |
| Historical Pattern Fit | 10 |
| Trap Detector | -10 deduction |

## Grade Bands

| Score | Grade | Action |
|---:|---|---|
| 90-100 | A+ | Premium bet |
| 80-89 | A | Strong bet |
| 70-79 | B | Small stake only |
| 60-69 | C | Very cautious |
| <60 | PASS | No bet |

## Key Principle

A team can have a high win probability but still receive a low betting grade if the market price is already too compressed.
